Automatically add image to WordPress post based on the title using duckduckgo engine.

== Description ==

This powerful plugin can save you both time and money, it can automatically insert an SEO optimized image based on the post title. The plugin uses duckduckgo engine to grab and display a related image.

== Installation ==

After installing this plugin you need a working CORS proxy server. There are several ways to get your own CORS anywhere server and you can [use Cloudflare to create your own server for free](https://www.example.com/blog/create-a-free-cors-proxy-server-on-cloudflare/ "use a free cloudflare CORS server"). Once you get your CORS proxy server link you can insert it in the plugin settings page.

= Minimum Requirements =

* WordPress 3.8 or greater
* PHP version 5.2.4 or greater

== Frequently Asked Questions ==

= How to install this plugin? =

Installation is very simple after downloading the plugin go to the settings page then add your CORS proxy server link and select which post categories allowed for the plugin.
